# Receipt Tracker
This app was created for the JHub Coding Scheme Module 2 challenge.

## Usage
As per the challenge instructions, a build release is not required so has not been included as I wish to make additions before an initial release.

The app is very self explanatory once installed. A user can create a trip and associate a number of receipts to this trip. Photos of receipts can be taken in app or selected from the gallery and notes may be added to each receipt.

All receipts associated to a trip or just individual receipts can be downloaded to a separate folder in the device's gallery. 

Once expenses have been claimed, a trip can be deleted.
